# 📦 Amazon Shipping Analytics Dashboard

A professional business intelligence dashboard built using Tableau to analyze Amazon shipping performance, order fulfillment trends, and shipment delays across categories and countries.

## 📌 Project Overview

The Amazon Shipping Analytics Dashboard provides actionable insights into shipping operations by tracking:

Total orders <br>
Shipped vs non-shipped orders <br>
Shipping performance trends over time <br>
Product category analysis <br>
Geographic distribution of delayed shipments <br>

This dashboard helps stakeholders monitor operational efficiency, identify shipping bottlenecks, and improve logistics decision-making.

## 📈 Key Insights
1. Shipping Performance <br>
- Total Orders: 74
- Non-Shipped Orders: 11
- Shipment Completion Rate: 85.14%
2. Category Analysis <br>
- Beverages and Dairy Products recorded the highest order volume.
- Confections showed a relatively higher percentage of non-shipped orders.
3. Time-Based Trends <br>
- Shipment activity remained stable throughout April 2020.
- Non-shipped orders increased toward the end of the month.
4. Geographic Insights <br>
- Delayed shipments were identified across multiple countries.
- North America showed the highest concentration of non-shipped orders.
